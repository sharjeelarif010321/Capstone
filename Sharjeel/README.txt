Within this directory you will find the following:

- Scripts created for the main algorithm.
- Test data for each of the four models trained based off test plan outlined for the model.
- Scripts used to train the model
- The trained model themselves
- Plots generated when training the machine learning models to track convergence
- Block diagrams and schematics for the algorithm 
- As well as deprecated scripts

Everything is updated as per 11th of April 2023.

For scripts that don't have a revision history, there has been a headerfile generated to track the revisions made.
